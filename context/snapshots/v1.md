## Architecture Baseline

### Tech Stack
- Next.js 14.2.x (App Router)
- TypeScript
- Redux Toolkit + RTK Query
- redux-persist
- Tailwind CSS 3
- SweetAlert2 (via lib/toast.ts wrappers only)
- react-hot-toast

### Route Count
- ~60 protected routes mapped in `lib/permissions.ts`
- 13 store setting tabs at `/store/setting`
- 15 report pages at `/reports/*`

### Redux Persisted Slices
- `auth` — user, token, currentStore, currentStoreId
- `invoice` — POS cart (per-store keyed)
- `orderEdit` — order being edited
- `orderReturn` — return/exchange in progress

### API Base
- `NEXT_PUBLIC_API_BASE_URL` — points to backend (default: http://127.0.0.1:8080)
- All endpoints go through `store/api/baseApi.ts`
- Bearer token injected automatically

### i18n Architecture
- Default language: Bengali (`bn`)
- Language detection: middleware → Vercel geo → `i18nextLng` cookie → `'bn'`
- `getTranslation()` works in both server components (reads `next/headers`) and client components (reads cookie)
